# AI Interview Prep — UI

## Setup

### Requirements
- Node.js 18+
- npm ya pnpm

### Steps

```bash
# 1. Dependencies install karo
npm install

# 2. Dev server start karo
npm run dev

# 3. Browser mein open karo
# http://localhost:5173
```

## Pages
- `/` — Dashboard
- `/interview` — Mock Interview
- `/chatbot` — AI Chatbot
- `/quiz` — Quiz
- `/profile` — Profile

## Build (Production)
```bash
npm run build
npm run preview
```
